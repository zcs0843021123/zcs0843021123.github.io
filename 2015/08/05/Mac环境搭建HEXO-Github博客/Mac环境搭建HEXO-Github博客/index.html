一. 安装工具

Hexo 官网指南 https://hexo.io/docs/index.html

  1. Git
  2. Node.js
  3. 安装hexo npm install -g hexo-cli
    注意：全局安装的npm第三方库在 /usr/local/lib/node_modules 里边,所以,安装hexo的结果可以在 /usr/local/lib/node_modules 中查看

  3. 安装node_modules第三方库

3. hexo主题

个人使用的是：iissnan/hexo-theme-next
源码地址在 https://github.com/iissnan/hexo-theme-next
next 官网 http://theme-next.iissnan.com/getting-started.html

  1. 下载主题

  2. 把 next 主题设为自己blog的主题, 修改 _config.yml 中的内容 theme: next

  3. 修改 _config.yml 中的内容 language: zh-Hans

  4. 创建一个 tags 文件
    hexo new page "tags"
    文件内容为:

    修改 themes/next/_config.yml 文件, 修改如下:

  5. 常用设置 (修改 themes/next/_config.yml 文件)

  6. 类似的, 添加 about (关于) 页面, 也是这么添加的。

  7. 在 source/_data 目录下创建一个 next.yml 文件 (如果没有 _data 就创建一个 _data 目录)

  8. 把 _config.yml 和 themes/next 下的 _config.yml 内容都粘贴到 next.yml 中

  9. 以后使用 hexo clean . hexo g . hexo s . hexo d 都要加上 --config source/_data/next.yml
    如: hexo clean --config source/_data/next.yml && hexo g --config source/_data/next.yml.

  10. 根据自己的喜好,再看着 next.yml 里的注释, 进行自己的喜欢的风格的配置。
    例如

    • 设置头像 可以设置 next.yml 中的 avatar 属性。

  11. 一定要注意,以后要是修改配置文件需要 themes/next下的_config.yml. _config.yml 与 next.yml 同时修改。
    当然要是某个文件里,压根就木有那个属性,我们就不用修改。

4. 插入图片

  1. 把主页配置文件 _config.yml 里的 post_asset_folder :这个选项设置为true
    把 next.yml 里的 post_asset_folder:这个选项设置为true
  2. 安装图片保存的第三方库 npm install hexo-asset-image --save
  3. 在 /source/_posts 里边造一个和 your_md_name.md 同名的目录 your_md_name,把图片放在里边。
  4. md文件中应用图片
    格式 ![你想输入的替代文字](图片所在的目录名/图片名.jpg)
    如 ![测试图片](Mac环境搭建HEXO-Github博客/Snip20170906_5.png)

四. 注意事项

  1. 注意事项 确保自己是 Hexo 3 (或者更高的版本)

  2. 有时候报错 ERROR Deployer not found: git 这时候,需要加装第三方库：

  3. 有时候在Safari上浏览 https://zcs0843021123.github.io 看不到,报错404；可能是浏览器的问题,可以用Chrome 或者手机试试,也可以清空缓存试试（Safari的清空缓存 在Safari的开发菜单下边）

  4. 推荐安装从 finder 直接点击叫出shell 的工具 Go2Shell (当然你也可以不安装 直接一步一步cd进去）
