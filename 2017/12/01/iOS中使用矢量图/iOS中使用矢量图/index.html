iOS中使用矢量图

发表于 2017-12-01

零. 写在前边的话

  1. 本系列文章是自己的学习总结,因此写的有些简略,如有任何问题,可以给我发邮件, 我的邮箱 1017295148@qq.com

一. 方法

1、png转SVG

inkscape工具

2、SVG转.ttf文件

A、登录阿里巴巴矢量图片库
B、点击图标管理
C、上传图标
D、放到暂存架
E、下载到本地，里边就有了矢量图对应的字体文件 .ttf 文件
              里边还有一个.html文件，里边有字体文件对应的实体字符码。
             详细信息都在HTML文件里边。

3、把iconfont.ttf 添加到工程目录下

4、在info.plist里边加上：

Fonts provided by application （它是一个数组）
item0: iconfont.ttf

5、Build Phase里边

CopyBundleResource的时候，把iconfont.ttf放进去，这样字体文件就被copy了，这样，字体文件就被安装上了。

6、使用时

UIFont *iconfont = [UIFont fontWithName:@"uxIconFont" size: 16];
    label.font = iconfont;
这个uxIconFont是字体名字，不是iconfont.ttf的名字。
label.text = @"\U00003439 \U000035ad \U000035ae \U000035af \U000035eb \U000035ec";

7、注意：

文本值为4个字节的Unicode码( UTF16 小端编码)，
所以，要在\U 与 字体值之间补上0。
13420/0x346c 这就是字体值，前边是十进制，后边是十六进制，我们使用的是十六进制。

8、使用BirdFont查看字体对应的编码，他可以直接打开.ttf文件。

A、打开BirdFont，打开字体文件（.tff文件）
B、选择All Glyphs

9、使用字体文件实现矢量图的原理：

就是造一个fontFamily
里边有一种特定的font
这种特定的font，它也是使用Unicode码，
只不过这种font，用我们自己定义的图案，代替了原本的字符。
