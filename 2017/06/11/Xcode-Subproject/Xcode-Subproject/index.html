  1. 本系列文章是自己的学习总结,因此写的有些简略,如有任何问题,可以给我发邮件, 我的邮箱 1017295148@qq.com

一.

1、子项目必须可以编译成 framework 或者是 静态库
2、新建一个项目 要求是 framework 或者是 静态库

3、新建项目 要编译通过

4、把子项目放到你的工程的目录里去，通过 Add files to project 添加子项目，千万不要自己拖拽，因为Xcode有BUG

5、Build Phases 中 Link Binary With Libraries 中 加上你的静态库

6、Build Settings 中 Header Search Paths 中 加上你的静态库所在的父目录 ZRIPSubProject

这里注意：
Header Search Paths 它只是寻找静态库或者是 framework 暴露出来的头文件，
你的头文件在哪，就写那个目录，（递归还是不递归，看情况），
Header Search Paths 是以 主项目 的 .xcodeproj 所在的目录为当前目录往下找的。
